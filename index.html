SharePoint Client Side Rendering

using only one tool for assembly

CSR = Client Side Rendering

transforms SharePoint output when displayed in the Browser (Client Side Rendering)

CSR can rewrite: the View, one List Item or one Column

CSR can be applies on: Views, and the Edit, New or Display Forms

JSLink

A JSLink specifies which JavaScript files are associated (executed) for a WebPart, View or Field

Yes, you can execute any JavaScript file, they do not have to contain CSR code.
