This Javascript code adds a JSLink Manager window to the current SharePoint ListView page.

With one click you can add (or delete) JSLink references of the current WebParts or Views of the current List.

Installation & Usage

Drag this Bookmarklet link: JSLinkManager to your Favourites bar

What is a Bookmarklet?

Open a SharePoint ListView page

Activate this bookmarklet from your Favourites bar

Technical details

When clicked the SLinkManager Bookmarklet adds this JavaScript file from our 365CSI server to your current SharePoint page; and executes the JavaScript code.
